## ADDED Requirements

### 需求: Memoir must remain a web-first asset studio
回忆录必须被定义为 Web 优先的深度整理与资产查看界面，系统不得要求首版 Native App 完整复制回忆录的长文本编辑、全量编排和来源核对能力。

#### 场景: 用户需要深度编辑回忆录
- **当** 用户需要修改章节内容、核对来源或进行较大范围的回忆录整理
- **那么** 系统必须允许这些深度操作以 Web 为主要界面完成
- **那么** 首版 Native App 不必完整提供相同级别的编辑器

#### 场景: 用户在 App 中只需要轻量回顾
- **当** 用户在 Native App 中查看近期回顾或某条记忆摘要
- **那么** 系统可以只提供轻量查看或跳转路径
- **那么** 系统不得因为首版 App 范围受限而改变回忆录仍然可追溯、可编辑的基础要求
