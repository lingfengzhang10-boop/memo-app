## ADDED Requirements

### 需求:分身必须记录同一话题的互动记忆
系统必须为分身记录轻量的话题互动记忆，以区分“第一次聊这个话题”和“这个 asker 刚刚已经问过这个话题”。

#### 场景:同一 asker 再次追问同一话题
- **当** 同一 asker 在短时间内再次询问同一 `topic_key`
- **那么** 系统必须读取该话题最近一次互动记录
- **那么** 系统必须将这次回答标记为递进、换角度或诚实收束，而不是默认 fresh answer

### 需求:互动记忆必须区分 same asker 与 different asker
系统必须把 same asker 的互动记忆与 different asker 的互动记忆隔离，禁止把上一位 asker 的对话痕迹直接复用于另一位 asker。

#### 场景:不同 asker 提问同一人生事实
- **当** 不同 asker 询问同一 topic_key
- **那么** 系统必须把这轮回答视为对当前 asker 的 fresh answer
- **那么** 系统不得使用“你之前问过”之类只适用于前一个 asker 的表达

### 需求:互动记忆必须具备自然衰减
系统必须根据距离最近一次讨论的时间间隔调整回忆强度，禁止永远精确地表现为“刚刚聊过”。

#### 场景:同一 asker 在不同时间间隔重问
- **当** same asker 重问同一 topic 且距离上次讨论时间处于不同区间
- **那么** 系统必须在 immediate、same_day、recent、stale 四种回忆强度之间选择一种
- **那么** 回忆强度必须影响回答措辞，但不得改变底层人生事实
